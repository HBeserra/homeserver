# Home Server LAB

Esse repositorio foi feito para manter os serviços que utilizo no meu home server, os serviços aqui utilizados não estao configurados para serem expostos diretamente a internet. 

**Gerenciamento:**
 - Proxmox
 - Portainer
 - DNS Server

**Monitoramento:**
  - Grafana
  - Loki
  - Prometheus

**Testes:**
  - k6
